Function: addGrpMbr
Description: Adds a member to a MOODLE group
llMessageLinked: llMessageLinked(LINK_SET, PLUGIN_CHANNEL, "plugin:user,function:addGrpMbr\nAWARDID:"+(string)currentAwardId+"\nSENDERUUID:"+(string)owner+"|GROUPNAME:"+currentGroup+"|USERUUID:"+(string)useruuid+"|USERNAME:"+userName, NULL_KEY);
Parameters:
AWARDID: sloodleid of award activity
SENDERUUID: uuid of user issuing command
GROUPNAME: Name of group to add member to
USERUUID: members uuid
USERNAME: avatar name of member
