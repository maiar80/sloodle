Sloodle Picture Glossary

The Sloodle Picture Gloss, provides an in-world glossary where users can search via chat commands. When a Glossary entry is found, the texture stored for that glossary entry is displayed on a Prim in Second Life.

Users to query a MOODLE Glossary by typing "/pix searchterm" The Picture will then searches the Moodle glossary for special formated Glossary entries which have Texture UUID's and scale dimensions stored. If a texture UUID has been stored as the Glossary terms definition, then the Picture Gloss in Second Life will be scaled to the appropriate dimension and it's textures changed to the Texture UUID stored in the Glossary entry.

The Video below gives a good example of what the Picture Gloss can do.


Setup

  1. To Use the Picture Gloss, you first need to create a Glossary activity in MOODLE.
  2. Next, add glossary entries for the textures you'd like to display in Second Life. For each glossary entry, add the following: textureUUID:scaleX:scaleY:scultpUUID

Example:

Next, in Second Life, rez a Picture Gloss tool, and configure it, either using the Notecard Configuration technique or the Web Setup technique. Once it is configured, and connected to your Glossary, users can view the textures stored by typing:

/pix name-of-glossary-entry

example

/pix Fire
