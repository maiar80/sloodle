SLOODLE Download Research Forums Documentation

If you have a problem with SLOODLE then please tell us what happened. We will try to help you find an immediate solution, and we'll aim to fix it in a subsequent release. Please be sure to tell us what you were trying to do, and what went wrong. Include any error messages that were reported, and please try to tell us what versions of Moodle and SLOODLE you are using.


Forums

The forums are a good place to report general problems. We can discuss the issue with you, and other people in the future can hopefully learn from it too. You will need to create an account on Sloodle.org in order to post a message:

  • SLOODLE Forums - use the "Discussion" forum for most purposes


Issues Tracker

If you are more experienced with software testing, then you might want to try using our Issues Tracker directly instead. It is fairly simple to use. Please remember to provide as much information as possible, and that we will also use the issues tracker for a lot of our discussion about solving the problem. If you want to be kept informed then just keep checking the comments on the issue.
