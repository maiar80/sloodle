Function: getGrps
Description: Returns a list of groups in the course }
llMessageLinked(LINK_SET, PLUGIN_CHANNEL, "plugin:lessons,function:getGrps\nSLOODLEID:null\n\INDEX:0|GROUPSPERPAGE:10", NULL_KEY);
Parameters:
SLOODLE_ID:null this is left as null because we aren't accessing a specific activity
INDEX: INDEX: Index of groups in course group list that you want to start returning output from
GROUPSPERPAGE: The MAX number of groups you want to return

/***********************************************
*  s()  used so that sending of linked messages is more readable by humans.  Ie: instead of sending a linked message as
*  GETDATA|50091bcd-d86d-3749-c8a2-055842b33484 
*  Context is added instead: COMMAND:GETDATA|PLAYERUUID:50091bcd-d86d-3749-c8a2-055842b33484
*  All this function does is strip off the text before the ":" char and return a string
***********************************************/
string s (string ss){
    return llList2String(llParseString2List(ss, [":"], []),1);
}
