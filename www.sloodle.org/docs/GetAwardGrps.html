Function: getAwardGrps
Description: Return all groups in the course and indicate which ones are connected with this award
llMessageLinked: llMessageLinked(LINK_SET, PLUGIN_CHANNEL, "PLUGIN:awards,FUNCTION:getAwardGrps\nAWARDID:"+(string)currentAwardId+"\nINDEX:"+(string)index_selectTeams+"|GROUPSPERPAGE:9", NULL_KEY);
Parameters:
AWARDID: sloodleid of award activity
INDEX: index of the award group we would like to start output from
GROUPSPERPAGE: How many groups we would like to include in our output
