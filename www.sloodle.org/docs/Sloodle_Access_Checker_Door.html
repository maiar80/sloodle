Works the same way as the Sloodle Access Checker, except that instead of being triggered by someone entering a space, it is triggered by being touched by an avatar. If it is touched by an avatar is registered with Sloodle and enrolled in the course, it opens to let them through.

This access checker door has been removed from Sloodle 1.0, simply because it actually acts as a Registration door, and does not actually prevent access to land areas in Second Life. No tool in Second Life can actually bar access. Once you are on an island, a user can simply utilizat their camera tools to view anywhere on the island. Thus, the access checker does NOT block access, only prevents an avatar from walking through the door if they are not a member of the course. The user could however, get around the door simply by using their camera tools, selecting a chair inside the room, and selecting "Sit"

Thus, we recommend this tool as a way to provide Registration, instead of blocking access.
