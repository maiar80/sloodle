Sloodle Versions: 0.2, 0.21 (?)

Please note: the information on this page refers to old version of Sloodle (0.2 and 0.21). The equivalent functionality and information for Sloodle 0.3 can be found using the Sloodle Distributor Module combined with the SLOODLE Vending Machine.


The Sloodle Object Distributor is not a usable object on its own. Instead, you need to use an interface, such as the web-based interface which is documented here. Alternatively, you can use an in-world interface, which is called the Sloodle Object Selector.


Configuration

There is no configuration required to use the Moodle distribution interface. You need to have a Sloodle Object Distributor rezzed in Second Life, and it needs to have connected successfully to your website. Anybody who has an account on your Moodle site can then access the distribution interface.


Where is it?

For educators/administrators, it is advisable to create a link somewhere on your Moodle site to the distribution interface, as it is a long address. You will find it at the following location, relative to the root address of your Moodle installation: 

/mod/sloodle/sl_distrib/sl_send_object.php

So for example, the distribution interface for Sloodle.org is at: 

http://www.sloodle.org/mod/sloodle/sl_distrib/sl_send_object.php


Usage

There are two parts of the interface, although most users will only see the first. Note also that, if the object distributor is not connected, or if it does not contain any items, then only an error message will be displayed.

Sections

"Send to me"

All users will be able to see this section, although it may be disabled. It allows a user to request the system to send an object to their avatar in-world. Usage is as follows:

  • Select the object you want from the drop-down menu
  • Click "Send to me"

Note that the "Send to me" button also contains your avatar's name, so you can confirm that it will be sent to the correct avatar.


This section will be disabled if the Moodle site doesn't know who your avatar is. You need to register (or 'authenticate') your avatar with the Moodle site, using an appropriate in-world object. Alternatively, you could simply use an in-world Sloodle Object Selector which is connected to the same site.


"Send to another avatar"

This section is only visible to Moodle users with Teacher or Administrator privileges, and it allows objects to be sent to any avatar which is registered with the Moodle site. Usage is as follows:

  • Select an object from the first drop-down menu
  • Select a recipient avatar from the second drop-down menu
  • Click "Send object"


Sending Objects

After you have clicked the appropriate "Send..." button, the system will attempt to contact the connected Distributor. This may take some time, so it is advisable to be patient. If nothing happens after a minute, or if an error message appears, then there is probably a problem with the distributor. It may need to be re-rezzed or re-connected.

WARNING! Rezzing more than one Distributor for the same site will cause interference. See Sloodle Object Distributor for more information.
