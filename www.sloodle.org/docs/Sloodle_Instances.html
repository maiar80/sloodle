What are instances?

Every Moodle site has lots of 'activity modules' installed, but you normally have to create an 'instance' of a module before you can use it. For example, you can have the forum module available on your Moodle site, but you have to add a forum to your course before you can use it. See the Moodle documentation for more information.


Can Sloodle have instances?

Yes! After a long wait, Sloodle 0.3 can now have instances. After Sloodle is installed, you must add a Sloodle Controller Module to each course where you wish to use Sloodle (this can be done by teachers). This module controls access and security, making your Moodle site safer in the event that a malicious user attempts to gain illegitimate access. It also gives the teachers much more freedom and flexibility.

The Sloodle module can also be used to create instances of the Sloodle Distributor Module, which works alongside the Sloodle Vending Machine object in-world. This functionality was previously available through the Moodle Distribution Interface and Sloodle Distributor, but they only allowed 1 distributor per Moodle site. With the Sloodle Distributor Module, you can have as many as you like.


Note: Sloodle versions prior to 0.3 cannot create instances, and they will not be updated to add this ability.
