Sloodle Documentation

As with many open source projects, Sloodle is a lot more mature than its documentation! However, we are working on this, and here we will build up a comprehensive list of documentation about all aspects of Sloodle, for everybody from users to developers.

  • Sloodle User Documentation - for Educators and Students
  • Sloodle Administrator Documentation - for anybody who needs to install/configure Sloodle
  • Sloodle Developer Documentation - for anybody who wants to contribute to Sloodle
  • Download Sloodle

What's the difference? There is some overlap of information between the categories, but only where necessary. You will find that Administrators will probably need to refer to the User Documentation as well, because it contains the information about using the components (whereas the Administrator Documentation only explains the installation and configuration of the Sloodle module).
